# tables/pilot_gse25055/

Output tables for the first GSE25055 pilot (branch
`pilot-leaky-vs-nested-gse25055`).

Generated by running the pilot scripts / notebook from the repository root:

- `pilot_performance_comparison.csv` -- one row per pipeline
  (`A_leaky_baseline` vs `B_guarded_nested`) with columns:
  `auroc`, `pr_auc`, `balanced_accuracy`, `mcc`, `sensitivity`, `specificity`.

This directory is committed empty (this README only) because the numeric
outputs require an R runtime with GEOquery + Biobase + e1071 + caret + pROC +
PRROC, which is not available in the commit environment. See
`results/pilot_gse25055/pilot_notes.md` for run status.
