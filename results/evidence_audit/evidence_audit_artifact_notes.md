# Reproducible Omics Evidence Audit — Artifact Notes

This artifact makes the manuscript's **Table 10 (Reproducible Omics Evidence Audit)**
reproducible and machine-readable. It contains **no new modeling**: every value is read from
outputs already committed to the repository, and the artifact is regenerated by a single script.

## Files
- `tables/evidence_audit/evidence_audit_schema.csv` — column dictionary for the final table
  (`domain, metric, value, interval, p_value, reference, interpretation, source`).
- `tables/evidence_audit/evidence_audit_final.csv` — the audit table, one row per metric, tagged
  with its audit domain and the committed source file it came from (27 rows across 10 domains).
- `tables/evidence_audit/evidence_audit_machine_readable.json` — the same content nested by domain,
  for programmatic consumption (e.g., a dossier/scorecard tool).
- `scripts/10_build_evidence_audit_artifact.R` — the canonical generator; reads the committed
  outputs and writes the three files above plus refreshes these notes.

## Ten audit domains
1. **Dataset integrity** — usable samples, class balance, platform, overlap.
2. **Leakage sensitivity** — internal leaky-minus-guarded gap (ΔAUROC, ΔPR-AUC) with CIs; DeLong as a secondary check.
3. **Bootstrap uncertainty** — 95% bootstrap CIs for leaky/guarded discrimination.
4. **Permutation control** — leaky vs guarded null distributions under shuffled labels (diagnostic of the leakage mechanism).
5. **Repeated-CV robustness** — per-seed gap across 30 seeds (median, range, Wilcoxon).
6. **Feature stability** — Nogueira index, Jaccard, stable-core/unstable-tail (single split and 30-seed).
7. **External transportability** — GSE25065 performance and the discovery→external drop.
8. **Class imbalance** — sensitivity/specificity and confusion counts under low pCR prevalence.
9. **Reproducibility status** — analysis scripts and committed outputs.
10. **Limitations / unresolved risks** — statistical resolution, generalization, clinical status.

## Regeneration
```
Rscript scripts/10_build_evidence_audit_artifact.R
```
The script reads only the committed outputs listed above and rewrites the schema CSV, final CSV,
JSON, and these notes. It performs no modeling, no resampling, and no external-data access.

## Interpretation guardrails
Values are reported as **within-cohort, diagnostic** evidence. In particular: the single-cohort
leakage gap is positive but its 95% CI includes zero; the 30-seed Wilcoxon result is fold-level
reproducibility, not patient-level inference; the permutation control demonstrates the leakage
*mechanism*, not the magnitude of the real-data gap; the external drop is a transportability limit,
not a leakage effect. This is a methodology/audit artifact, **not** clinical biomarker discovery.
